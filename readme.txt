== Description ==
Schedule tags allows site admin to set post tags for the given time frame. So that when a schedule set time frame comes the plugin automatically sets tags for the post and when the schedule time expires the plugin silently removes the tags from the post.


== Installation ==
1. Login to your WordPress dashboard and navigate to Plugins > Add New
2. Search for Schedule Tags.
3. Click install.
4. Click Activate.
5. Once the plugin is installed then you can see Schedule Tags in the admin settings.
